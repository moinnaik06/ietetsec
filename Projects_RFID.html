RFID Treasure Box

About RFID Treasure Box


HARDWARE

RFID stands for radio-frequency identification which uses electromagnetic fields to automatically identify and track tags connected to objects.Such identifiaction cards are often used in organised companies.

Our RFID treasure box is a very simple,luck based game.The player is presented a choice of 5 cards,one of which has the RFID tag that unlocks the box which holds the prize.If the player chooses the right one, the box unlocks and they win.

Components


RFID Cards

Contain the RFID tags which unlock the box.


Arduino Uno

It contains the code and powers the Led's


Servo

To lock/unlock the box.


Wires

To make connections.
