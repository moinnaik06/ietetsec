OUR PROJECTS

These projects were made by our hardworking team

    • OBSTACLE AVOIDING CAR

    Hardware

    Built on an arduino base,this car can avoid any obstacle in its path and turn in a different direction.We used sensors for obstacle detection and servo motors for wheels along with a power bank to power the vehicle

    • ZOMBIE GAME

    Software

    'Temple Run' like games have been a huge hit in the app market lately.This game,designed by our team using unity is played using an x-box controller and a laptop..It is built using unity and is windows as well as android based.

    • LED ROULETTE

    Hardware

    LED roulette is a fun and easy game which checks our reflexes.It is very simple,click the button when the RED LED is glowing and you win.It has a small and easy structure based on Arduino.It is a fun way to pass time

    • Drone Racing Game

    Software

    Another fun and easy game which is also competitive and exciting.It is made on a unity platform and has amazing graphics with a map to navigate through the game.Find the portal and discover amazing gifts.

    • RFID Treasure Box

    Hardware

    RFID treasure box is another simple luck based game we developed using arduino.It uses an RFID lock which can only be opened by the right card.Choose true and win an amazing prize.

    • AR GAME

    Software

    Augumented reality games like pokemon go are getting popular with each passing day.Our game is simple and luck based, choose a card and hold it in front of the camera, if the card is right you win.
